DOCUMENTAZIONE CAMPI DI INPUT
=============================



  Componenti form per l'input di dati
  ======================================


    ButtonForm
    =================
    Componente di input per la selezione di un valore preimpostato

      import
      ======
      import { ButtonForm } from '{path}/index.form';

      Props
      =====
      children, // nodo figlio del componente
      name, // nome del component
      className, // classi di stile del component
      value, // valore tornato dal componente
      onClick, // evento eseguito al click del mouse sul componente; ritorna, nell'evento, il valore di value
      ... // other props: React "button" component props (type="button")

      Default CSS class
      .input .input__button

  Componenti form di layout
  ======================================


    Field
    =================
    Contenitore dei campi di input;

      import
      ======
      import { Field } from '{path}/index.form';

      Props
      =====
      children, // nodo figlio del componente
      label, // stringa di testo da visualizzare nel componente "label"
      className, // classi di stile del component

      .field
